# 칼렌다 컴포넌트.

nextjs 프로젝트를 생성하고, 퍼블이 넘겨줄 만한 내용만 적용한 상태입니다.  
프로젝트를 다운 받은 후, 프로젝트 디렉토리에서,
```shell
npm install
```
을 실행하여 필요한 패키지를 다운로드 후,
```shell
npm run dev
```
로 실행합니다.
